贡献

我们已经使用 GitHub 托管 LCUI 项目，您可点击此处进行查看。

以下是 LCUI 项目在后续版本更新中需要完成的任务，若您比较了解以下某个任务所涉及到的相关知识，您可以尝试完成这个任务。

未来将要完成的任务
分类任务说明
输入处理
  1. 完善鼠标输入处理，使之支持滚轮操作。
  2. 完善按键输入处理，能够支持特殊按键，例如：ctrl，shift，alt等按键，并提供类似以下原型的函数：
    int key_pressed(int key_code); 用于检测指定键值的按键是否处于按下状态。
    int key_free(int key_code); 用于检测指定键值的按键是否处于已释放状态。
GUI部件
  1. 新增文本框（TextBox）部件，用于实现数据录入功能。
    至于实现方法，可参考其它GUI已实现出来的文本框的效果，分析实现原理，并用代码模拟实现出来。
  2. 新增列表框（ListBox）部件，需要具备排序功能，可根据选中列中的数据进行升序或降序排列。
  3. 新增滚动条（ScrollBar）部件，容器内的内容超出显示范围时，可用滚动条来浏览未显示的内容。
待到以上两个部件实现之后，会启动另一个项目：文件管理器，计划在2012年年底之前发布第一个版本。
2D图形处理
  1. 支持bmp，jpeg文件的创建。
  2. 支持gif文件的读取，可正常播放读取出来的动画。
  3. 提供类似以下原型的函数，能为图形绘制阴影，可选择光照射的角度，阴影的大小，以及阴影的距离。
    int draw_graph_shadow ( LCUI_Graph *graph, int size, int light_angle, int Distance );
    此功能主要是为了实现文字的阴影、窗口/部件的阴影。
  4. 提供类似以下原型的函数，能将图形圆角化，也就是将矩形图形的四个直角变为圆角，可通过调整半径来调整圆角化的程度。
    int round_angles ( LCUI_Graph *graph, int radius );
  5. 提供类似以下原型的函数，能为图形描边，可指定边线的大小，能为图形边缘绘制不同粗细的线条，线条颜色也可以自定义，此函数可为字体描边。
    int draw_edge_line ( LCUI_Graph *graph, int size, LCUI_RGB color );
字体处理 目前对 微软雅黑 字体的支持还比较好，其它字体会存在部分问题，例如：宋体，在某些字体尺寸下，获取的汉字位图有问题，没有成型的汉字。需要解决。
音频处理 支持wav，mp3格式的音频文件的解码，支持音频输出，可混音播放，可调节音量大小。
应用程序 开发一些简单的小游戏，例如：扫雷、贪吃蛇、空当接龙、俄罗斯方块等；或者小软件，例如：画图。
文本框部件实现后，输入法也需要考虑，尤其是拼音输入法，需要设计一个搜索引擎，以及存储汉字信息的数据库，效率要求也很高，不然的话，输入一个拼音字母都要等几秒，那谁还敢用？ 看有什么开源的输入法可以用用。
其它 获取电量信息。
