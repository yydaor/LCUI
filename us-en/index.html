LCUI Homepage

Welcome to LCUI Homepage, here are provide reliable official LCUI version as well as related support.

LCUI is an open-source GUI library on the GNU / Linux platform, source code released under the GPLv2 license.

We are working hard to make it become a mature, stable, efficient GUI library，open source, meaning that anyone can participate in the development of this project, and jointly promote the LCUI development.

If you need run LCUI program, you must switch to console mode.

Download & Install

On the right to download the latest version of LCUI.

Download source code package to local, decompress package, enter to source code directory, read the README file to view LCUI introduction, installation method.

If you found in the course of the project exist BUG, try to write a small program that can reproduce the problem, and send the corresponding source code to our e-mail.

We has been used the GitHub to hosting LCUI project, you can access the GitHub website at the right, welcome to participate LCUI's optimization and improvement.
